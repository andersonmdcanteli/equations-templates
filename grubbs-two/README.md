# Grubbs Two ( $G^{''}$ )

$$ G^{''}=\frac{x_{n} - x_{1}}{s}$$

onde:
- $x_{1}$ é a observação com menor valor;
- $x_{n}$ é a observação com maior valor;
- $s$ é o desvio padrão amostral;
- $x_{1}, x_{2}, ..., x_{n}$ são os valores da amostra ordenados de forma crescente;
