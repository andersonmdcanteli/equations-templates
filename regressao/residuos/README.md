# Resíduos ($residuos$)

$$resíduo_{i} = y_{i} - \hat{y_{i}}$$

onde:
* $y_{i}$ é a i-ésima observação da variável dependente;
* $\hat{y_{i}}$ é o valor predito pelo modelo para a i-ésima observação da variável dependente;
