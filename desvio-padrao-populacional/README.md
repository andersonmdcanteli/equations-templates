# Desvio padrão populacional ( $\sigma$ )

$$\sigma = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (X_{i} - \mu)^{2}} $$

onde:
- $N$ é o tamanho da população;
- $\mu$ é a média da população;
- $X_{i}$ é a i-ésima observação da população;

## Código

# Variância populacional ( $\sigma^{2}$ )
