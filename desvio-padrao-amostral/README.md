# Desvio padrão amostral ( $s$ )

$$s = \sqrt{\frac{1}{\left(n-1\right)} \sum_{i=1}^{n}  \left(x_{i} - \overline{x}\right)^{2}}$$

onde:
- $n$ é o número de observações da amostra;
- $\overline{x}$ é a média da amostra;
- $x_{i}$ é a i-ésima observação da amostra;
